# Weather API Documentation

## Overview

Weather API is a fictional API designed to provide real-time weather data, weather forecast and weather alerts for developers weather-enabled applications.

This repository demonstartes professional API documentation practices using Markdown and Github.

## Features
* current weather
* 7-Day forecast 
* City Search
* Authentication 
* JSON Responses
